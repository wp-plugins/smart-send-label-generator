Automated shipping for WooCommerce. Generate shipping labels for Post Danmark (PacsoftOnline), GLS, SwipBox and Bring directly from WooCommerce.

== Description ==

The Smart Send label module makes it easy to print parcel labels directly in your webshop.
Labels can be generated in two ways; directly from the order detail site, or with the bulk action from the order list.

It's free to try for 30 days. Sign up on: [SmartSend.dk](http://smartsend.dk)

You can use multiple carriers simultaneous. The following carriers is supported
*   Post Danmark - Pacsoft Online
*   GLS - YourGLS
*   SwipBox
*   Bring - MyBring

In the module settings you can choose the carrier you want to bind with different kinds of shipping methods.

To create usable labels one must have a Smart Send subscription with valid carrier credentials. Demo labels that cannot me used for real packages can be generated using the “demo” account.

[youtube http://www.youtube.com/watch?v=jEvGbYy1sCo]

== Installation ==

1. Upload the entire 'smartsend’ folder to the '/wp-content/plugins/' directory. Alternatively upload the zip under “upload plugin” in WordPress.
2. Activate the plugin through the 'Plugins' menu in WordPress

1. Download and install the module/plugin.
2. Insert username and key
3. Add demo as KEY and Password in module setting if you don’t have a Smart Send user yet. If you do have a Smart Send user please insert the Key and password you received via e-mail.
4. Choose which carriers should be used for the different shipping methods.
5. You are now ready to print labels.

You can now print labels directly from within your shop and save a huge amount of time every day.

== Frequently Asked Questions ==

= Can I use the module with pickup points =

Yes. But only with the Pickup point module from Smart Send.

= Can I try the module without subscribing at Smart Send =

Yes. First time you try the module we create a test account for you that is enabled for 30 days.
